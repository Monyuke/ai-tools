# Edit List

streamlit用コンポーネント。

- 引数
  - List[EditData]
- EditDataのリストから単体Editコンポーネントを表示する

## 単体Editコンポーネント

- 引数
  - EditData
- EditDataの中身を表示
  - 中身はそれぞれ編集可能にする
- 適用ボタン
  - このEditDataを実際に適用する。
  - 適用後は適用済みと表示し、ボタンは非表示にする。
